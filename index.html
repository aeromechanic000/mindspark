📖 Usage Guide

🖱️ Mouse Controls

  • 🖱️ Click Select node
  • 🖱️ Double-click Edit text
  • 🖱️ Drag Move node
  • 🖱️ Right-click Add child node
  • 🖱️ Wheel Zoom in/out
  • 🖱️ Drag background Pan view

⌨️ Keyboard Shortcuts

  • ⌨️ Space Add child to selected
  • ⌨️ Delete Remove selected node
  • ⌨️ Enter Finish editing
  • ⌨️ Escape Cancel editing

✨ Features

  • Customize nodes with colors, shapes, and sizes
  • Save and load mindmaps as JSON files
  • Generate mindmaps using AI prompts
  • Interactive zoom and pan functionality
  • Real-time node editing and organization
  • Responsive design for all devices

ℹ️ About

Mindspark is a powerful, interactive mindmapping tool that helps you visualize and organize your ideas. Create beautiful, connected diagrams that represent your thoughts and concepts with ease.
